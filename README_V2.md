## 🎉 Новые функции v2.0:

### 1️⃣ 📈 График прибыли
**Кнопка:** `📈 График` (в меню бота)

Красивые графики с matplotlib:
- Накопленная прибыль (зелёная линия)
- Ежедневные столбцы прибыли
- Автоформатирование дат
- Тёмная тема

---

### 2️⃣ 🤖 AI-Советник
**Кнопка:** `🤖 AI Анализ` (в меню бота)

ChatGPT анализирует бота:
- Оценка производительности
- Рекомендации (start/stop/continue)
- Анализ рисков
- Работает БЕЗ ключа (mock режим)

**С API ключом OpenAI:**
```env
OPENAI_API_KEY=sk-your-key
```

---

### 3️⃣ 📋 Копирование бота
**Кнопка:** `📋 Дублировать` (в меню бота)

Копирует за 2 секунды:
- ✅ Символ (BTCUSDT)
- ✅ Сумму ордера
- ✅ Режим работы
- ✅ API ключи

---

### 4️⃣ 🎨 Темы оформления
**Кнопка:** `⚙️ Настройки` → `🎨 Тема`

Выбор темы:
- 🌙 Тёмная
- ☀️ Светлая
- 💜 Фиолетовая
- 🌊 Синяя

---

### 5️⃣ ⏰ Дневной отчёт
**Кнопка:** `⚙️ Настройки` → `⏰ Дневной отчёт`

Автоотчёт каждый день в 00:00:
- 💰 Прибыль за день
- 📈 Количество сделок
- 🏆 Лучшая сделка
- 📊 Сравнение с вчера

## 🐛 Troubleshooting:

### График не работает:
```bash
pip install matplotlib --break-system-packages
```

### AI Advisor не отвечает:
Работает в mock режиме без API ключа.  
Для реального AI: добавьте `OPENAI_API_KEY` в `.env`

### Бот не запускается:
```bash
python --version  # Должен быть 3.8+
pip install -r requirements.txt --break-system-packages
python main.py
```
