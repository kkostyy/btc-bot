# 🎉 Новые функции BTC Trading Bot v2.0

## ✨ 3 новые функции:

### 1️⃣ 📈 График прибыли
**Кнопка:** `📈 График` (в меню бота)

**Что показывает:**
- 📊 График накопленной прибыли (зелёная линия)
- 📊 Столбцы ежедневной прибыли (зелёные/красные)
- 📅 Автоматическое форматирование дат
- 🎨 Тёмная тема

**Как использовать:**
1. Выберите бота
2. Нажмите **📈 График**
3. Получите красивый график

---

### 2️⃣ 📋 Дублирование бота
**Кнопка:** `📋 Дублировать` (в меню бота)

**Что копируется:**
- ✅ Символ (BTCUSDT)
- ✅ Размер ордера
- ✅ Режим работы
- ✅ API ключи Binance

**НЕ копируется:**
- ❌ Баланс (начинается с $0)
- ❌ История
- ❌ Статус (stopped)

**Использование:**
1. Настроили один бот идеально
2. Нажали **📋 Дублировать**
3. Получили копию за 2 секунды!
4. Пополнили и запустили

---

### 3️⃣ 🎨 Темы оформления
**Кнопка:** `⚙️ Настройки` → `🎨 Тема`

**Доступные темы:**
- 🌙 **Тёмная** (по умолчанию)
- ☀️ **Светлая**
- 💜 **Фиолетовая**
- 🌊 **Синяя**
