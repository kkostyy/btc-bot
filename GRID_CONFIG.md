# 📊 Настройки сетки ордеров - ОРИГИНАЛЬНАЯ ВЕРСИЯ

## ✅ Текущая конфигурация

### Параметры сетки:
- **Начальный отступ:** 0.175% от центральной цены
- **BUY шаг:** 0.075% вниз между уровнями
- **SELL шаг:** 0.095% вверх между уровнями
- **Количество уровней:** 5 BUY + 5 SELL = 10 ордеров

---

## 📐 Математика сетки

### Формулы:
```python
# Начальные цены (первый уровень)
BUY_1  = center_price * (1 - 0.00175)  # -0.175%
SELL_1 = center_price * (1 + 0.00175)  # +0.175%

# Последующие уровни
BUY_N  = BUY_(N-1)  * (1 - 0.00075)    # каждый -0.075%
SELL_N = SELL_(N-1) * (1 + 0.00095)    # каждый +0.095%
```

## 🎯 Когда эта сетка оптимальна:

✅ **Флетовый рынок** (боковое движение ±1-2%)
✅ **Средняя волатильность** (обычные торговые дни)
✅ **Баланс риск/прибыль** (не слишком близко, не слишком далеко)
✅ **Стабильная торговля** (регулярные закрытия сделок)

## 🔧 Как изменить сетку:

### Сделать более агрессивной (больше сделок):
```python
# Уменьшить начальный отступ
buy_price = center_price * (1 - 0.00100)   # -0.1% вместо -0.175%
sell_price = center_price * (1 + 0.00100)  # +0.1%

# Уменьшить шаги
buy_price *= (1 - 0.00050)   # -0.05%
sell_price *= (1 + 0.00050)  # +0.05%
```

### Сделать более консервативной (меньше риска):
```python
# Увеличить начальный отступ
buy_price = center_price * (1 - 0.00250)   # -0.25%
sell_price = center_price * (1 + 0.00250)  # +0.25%

# Увеличить шаги
buy_price *= (1 - 0.00100)   # -0.1%
sell_price *= (1 + 0.00120)  # +0.12%
```
